## Overview

Data analysis requires clear questions, reproducible steps, and honest interpretation.

## Workflow

1. Define the analysis question before touching the data.
2. Inspect the data: shape, types, missing values, distributions, outliers.
3. Clean data: handle nulls, fix types, normalize formats.
4. Apply `exploratory-data-analysis` for initial exploration.
5. Use `scientific-visualization` for plotting and `statistical-analysis` for significance testing.
6. Document assumptions, transformations, and decisions.
7. Interpret results honestly — acknowledge limitations and alternative explanations.

## Quality Gate

An analysis is complete when the question is answered, assumptions are documented, and the process is reproducible.
