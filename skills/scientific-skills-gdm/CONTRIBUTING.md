# Contributing to Science Skills

Thank you for your interest in the Science Skills repository!

## Our Contribution Policy

At this time, we are not accepting external pull requests or code contributions.

## How You Can Help

Your feedback is invaluable for improving the quality of these skills. For any
of the below, please
[open an issue](https://github.com/google-deepmind/science-skills/issues):

-   API endpoint breakages, authentication failures, or rate-limit anomalies.
-   Execution bugs within the provided .py or .sh wrapper scripts.
-   Documentation typos or broken links.
