<objective>
Surface skill events to users via concierge messaging and heartbeat digests, and add the CLI skill management interface.

Purpose: This plan completes the user-facing layer of skill discovery. Users learn about new skills through natural concierge messages, see promotions in heartbeat digests, and can manage their skill library via `tamux skill` CLI. This is what makes skill discovery feel like the agent is "growing" (per the CONTEXT.md specifics).

Output: Concierge skill announcements, HeartbeatDigest skill events, IPC protocol extensions, CLI `tamux skill` subcommand, and server-side handlers.
</objective>
