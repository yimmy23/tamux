**Helper function `find_sibling_binary`:**
Add this near the top of main.rs (after the use statements):
```rust
/// Find a binary by checking the same directory as the current executable first,
/// then falling back to the bare name (which uses PATH lookup).
fn find_sibling_binary(name: &str) -> std::path::PathBuf {
    if let Ok(current) = std::env::current_exe() {
        if let Some(dir) = current.parent() {
            let sibling = dir.join(name);
            if sibling.exists() {
                return sibling;
            }
        }
    }
    // Fall back to bare name -- OS will search PATH
    std::path::PathBuf::from(name)
}
```

Do NOT add `which` as a dependency -- the fallback to bare name already triggers PATH lookup when spawned via `Command::new()`.
