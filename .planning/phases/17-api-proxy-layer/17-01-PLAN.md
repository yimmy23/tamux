         - `urlencode`: uses `percent_encoding::utf8_percent_encode(&s, percent_encoding::NON_ALPHANUMERIC)` to encode string values. Use `handlebars_helper!` macro per D-02.

       - `pub fn render_request(registry: &Handlebars, api: &ApiSection, endpoint: &EndpointDef, context: &serde_json::Value) -> Result<super::api_proxy::RenderedRequest, super::api_proxy::PluginApiError>`:

         - Renders each header value in `endpoint.headers` via Handlebars

         - Uses endpoint.method as-is (GET, POST, PUT, DELETE, PATCH)
         - Wraps rendering in `tokio::task::spawn_blocking` + `tokio::time::timeout(Duration::from_secs(1), ...)` per D-03
       - `pub fn render_response(registry: &Handlebars, endpoint: &EndpointDef, response_json: &serde_json::Value) -> Result<String, super::api_proxy::PluginApiError>`:
         - If `endpoint.response_template` is Some, register it as a template and render with response_json as context
         - If None, return `serde_json::to_string_pretty(response_json)` per D-05
       - Inline `#[cfg(test)] mod tests` with tests for each helper, context building, request rendering, response rendering, strict mode rejection, and timeout behavior
